Pre-requisite : Visual Studio IDE
1. Extract the project
2. Open project folder in file explorer and double-click solution file named as "Ensek.sln"
3. Once project is loaded in Visual Studio
4. Click "Build" menu in menu bar and chose "build solution" option.
5. Then click "Test" menu in menu bar and choose "Test Explorer" option, it will open ""Test Explorer"" tab.
6. In Test Explorer Dropdown "Ensek > Ensek.Features > BuyEnergyFeature"
7. Right-click on "BuyEnergyFeature" and choose "Run" option
8. You will see the Test execution running in Chrome browser.
